## About
This is a sandboxed command line interface for iOS. 


Commands included:

|	| 	|  | |
| ------------- |-------------| -----| -----|
|cat | cd | chflags | chksum |
| clear | compress | cp | curl |
| date | du | egrep | fgrep |
| grep | gunzip | gzip | help |
| link | ln | ls | mkdir |
| mv | printenv | readlink | rm |
| rmdir | stat | sum | tar |
| touch | uname | uncompress | uptime |
| wc | whoami | | |
